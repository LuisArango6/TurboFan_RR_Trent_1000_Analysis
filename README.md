# TurboFan_RR_Trent_1000_Analysis
Analysis of a Rolls-Royce Trent 1000 turbofan model. It includes the complete stage-by-stage analysis of both the fan and the core at take-off (2600 m ASL) and at cruise (11000 m ASL). The objective is to provide the thermodynamic analysis using Python, where each stage, the factors that influence it, and the results obtained can be clearly seen.
